# WP Posts Wrapper Contracts

Contracts and wrappers for WordPress Posts API.

## Supported Functions

- `get_post()`
- `get_posts()`
- `wp_insert_post()`
- `wp_update_post()`
- `wp_delete_post()`
- `wp_trash_post()`
- `wp_untrash_post()`
- `get_post_meta()`
- `update_post_meta()`
- `add_post_meta()`
- `delete_post_meta()`
- `get_post_field()`
- `get_post_status()`
- `get_post_type()`
- `register_post_type()`
- `unregister_post_type()`
